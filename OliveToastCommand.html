정규표현식

커맨드 질문에 일반 텍스트뿐만이 아닌 정규표현식을 사용할 수 있습니다.

+커맨드 만들기 "\+이메일확인 [0-9a-zA-Z]+@[0-9a-zA-Z]+.[0-9a-zA-Z]+" 이메일주소네요

=> +이메일 확인 sans@sans.com
-> 이메일주소네요

정규표현식 외에도 일반 텍스트를 사용할 수 있지만, 정규표현식 오류를 방지하기 위해서 특수문자에 경우 이스케이프 문자로 방지해주세요

+커맨드 만들기 \+와 샌즈!

=> +와
-> 샌즈

만약 식 맨 앞에 ^가 없거나 또는 $가 없다면 자동으로 입력됩니다.

자세한 정규표현식에 대한 내용은 여기서 참고해주세요.

arg

커맨드에 arg를 넣을 수 있습니다.

+커맨드 만들기 "와! (.+)!" {1} 아시는구나!

=> 와! 언더테일!
-> 언더테일 아시는구나!

=> 와! 올리브토스트!
-> 올리브토스트 아시는구나!

{0} 은 메시지 전체 내용입니다.

계산

계산 식을 사용할 수 있습니다.
이 계산 식은 올리브토스트의 +계산 커맨드와 같습니다.

+커맨드 만들기 1더하기1이뭐야\? <1+1>야.

=> +테스트
-> 2

자세한 계산식에 대한 내용은 여기서 참고해주세요.

변수

모든 변수는 [유저.아이디] [메시지.반응추가 sans]의 꼴로 사용할 수 있습니다.

메시지

삭제

메시지를 삭제합니다.


아이디

메시지의 아이디를 가져옵니다.


시간

메시지의 작성 시간을 가져옵니다.


고정

메시지를 채널에 고정합니다.


반응추가 (이모지이름)

메시지에 반응을 추가합니다.


유저

유저네임

유저의 디스코드 닉네임을 가져옵니다.


닉네임

유저의 서버 닉네임을 가져옵니다.
만약 서버 닉네임이 없다면 디스코드 닉네임을 가져옵니다.


핑

유저를 핑(멘션) 합니다.


아이디

유저의 아이디를 가져옵니다.


식별자

유저의 식별자(태그)를 가져옵니다.


상태

유저의 상태를 가져옵니다.


활동.타입

유저가 하는 활동의 타입을 가져옵니다.


활동.이름

유저가 하는 활동의 이름을 가져옵니다.


봇?

유저가 봇인지 판단하고 true, false로 리턴합니다.


가입일

유저가 디스코드에 가입한 시간을 가져옵니다.


입장일

유저가 서버에 입장한 시간을 가져옵니다.


추방

유저를 추방합니다.


차단

유저를 차단합니다.


역할추가 (역할이름)

유저에게 역할을 추가합니다.


역할제거 (역할이름)

유저에게서 역할을 제거합니다.


메시지 (내용)

유저에게 올리브토스트가 메시지를 전달합니다.

이름

서버
